Como criar uma variável de ambiente no heroku : 
heroku config:set MESSAGE="variáveis funcionam" --app nome do seu app

para ver os logs:
heroku logs --app noem do seu app

Qual o nome do app?
Resposta o link que termina em .com no heroku 

como criar um novo ambiente:

heroku create --remote prod(pode ser qualquer nome)

deploy: git push nome do remote
